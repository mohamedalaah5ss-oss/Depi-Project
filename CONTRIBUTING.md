# Contributing

Thank you for your interest in this project!

## Project Scope

This repository is a **solo graduation/portfolio project** (DEPI Data Engineering track). It is not currently open for feature contributions or pull requests.

However, you are welcome to:
- ⭐ Star the repository if you find the architecture useful as a reference
- 🍴 Fork it and adapt it for your own learning
- 🐛 Open an Issue if you spot a factual error in the documentation, a broken link, or a security concern
